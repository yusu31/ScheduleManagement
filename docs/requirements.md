# 要件定義書

## アプリ概要

**アプリ名：** 福島イベントナビ（仮）  
**リポジトリ名：** EventFinder

### 一言説明

福島県内（郡山市・本宮市・いわき市など）のイベント情報を自動集約し、  
地域住民や観光客が「今ここで何が起きているか」をひと目で把握できる地域密着型 Web アプリ。

### 背景・目的

- 福島県内のイベント情報が複数サイトに分散しており、まとめて確認できる場所がない
- 特に IT 系イベント（勉強会・ハッカソン）は Connpass に掲載されているが、福島で絞り込む手段が限られている
- 「地域 × 日付 × カテゴリ」で素早く検索できるツールがあれば、参加者・主催者ともに便利

---

## ターゲットユーザー

| ユーザー | 利用シーン |
|---------|-----------|
| 福島県内の住民 | 近くで開催されるイベントを探したい |
| 県外からの観光客 | 旅行中に参加できるイベントを探したい |
| イベント主催者 | 自分のイベントを登録して告知したい |
| IT エンジニア・学習者 | 福島の勉強会・もくもく会を探したい |

---

## MVP 機能（最優先で実装するもの）

優先度が高い順に並べる。

| # | 機能 | 説明 |
|---|------|------|
| 1 | イベント一覧表示 | 登録済みイベントを一覧で表示する |
| 2 | イベント詳細表示 | イベントの詳細情報を表示する |
| 3 | イベント登録 | ユーザーが新しいイベントを登録できる |
| 4 | イベント編集 | 登録したイベントを編集できる |
| 5 | イベント削除 | 登録したイベントを削除できる |
| 6 | 絞り込み検索 | 地域・カテゴリ・日付でフィルタリングできる |
| 7 | カレンダー表示 | イベントをカレンダー形式で表示できる |
| 8 | ユーザー認証 | サインアップ・ログイン・ログアウト |
| 9 | Connpass 連携 | Connpass API から福島関連イベントを自動取得・保存 |
| 10 | お気に入り | ログイン後、イベントをお気に入りに追加できる |

## 非機能要件

| 項目 | 内容 |
|------|------|
| レスポンシブ | スマートフォン・タブレット・PC すべてで使える |
| セキュリティ | パスワードはハッシュ化して保存（Devise のデフォルト機能を使用） |
| データ取得 | Connpass からの自動取得は手動実行（定期実行は MVP 後に検討） |
| スクレイピング禁止 | 外部データ取得は Connpass 公式 API のみ使用する |

---

## 制約・注意事項

- Web スクレイピングは使用しない（法的リスクのため）
- 外部データ取得は Connpass 公式 API のみ
- MVP 完成後に追加機能（地図表示・プッシュ通知など）を検討する
